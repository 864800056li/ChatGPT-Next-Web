# 🏦 金融贷款咨询自动回复系统 - 常见问题

## 一、基础问题

### Q1：这个系统能做什么？
**A**：自动回复微信/企微上的贷款咨询，24小时在线回答客户关于利率、额度、要求、流程等标准化问题，筛选高意向客户转人工跟进。

### Q2：需要多少技术基础？
**A**：零基础即可使用。只需：
1. 执行一条防睡眠命令（或运行一键脚本）
2. 填写3个常见问答
3. 运行演示脚本看效果

### Q3：会不会封号？
**A**：
- **企业微信**：零风险，官方API
- **个人微信**：有风险，建议先用小号测试
- **推荐方案**：使用企业微信，合规安全

## 二、安装与配置

### Q4：防睡眠命令执行失败？
**A**：可能原因：
1. **密码错误**：输密码时不显示，正常现象
2. **权限不足**：确保是管理员账户
3. **终端问题**：尝试新开一个终端窗口

**备选方案**：
```bash
# 简版命令（通常够用）
sudo pmset -c sleep 0 displaysleep 0
```

### Q5：如何填写业务问答模板？
**A**：打开 `data_loan_template.txt`，填写：
1. 公司基本信息
2. **至少前3个问题**（利率、额度、要求）
3. 用你实际跟客户说的语言
4. 确保合规准确

**示例**：
```
客户问题：利率是多少？
你的专业回答：您好！年化利率4.5%-9.6%，根据征信、收入综合评估。
```

### Q6：演示脚本运行不了？
**A**：确保在正确目录：
```bash
cd ~/.openclaw/workspace/wechat-auto
python demo_loan.py
```

如果提示"python命令不存在"，尝试：
```bash
python3 demo_loan.py
```

## 三、业务相关

### Q7：贷款业务回复需要注意什么？
**A**：
1. **合规性**：符合金融监管要求
2. **准确性**：利率、额度等数据准确
3. **风险提示**：必须包含风险提示
4. **专业术语**：使用行业标准术语

### Q8：如何设置自动筛选客户？
**A**：系统可识别客户意向关键词：
- **高意向**："急需"、"马上"、"确定要"
- **中意向**："考虑"、"比较"、"了解一下"
- **低意向**："随便问问"、"好奇"

高意向客户自动标记，提醒人工跟进。

### Q9：能处理多少种问题？
**A**：系统预设24种贷款相关问题：
- 利率利息（4种）
- 额度金额（3种）
- 期限还款（3种）
- 申请要求（3种）
- 办理流程（3种）
- 征信信用（3种）
- 客服联系（3种）
- 安全保障（2种）

可根据你的业务扩展。

## 四、进阶功能

### Q10：如何集成到企业微信？
**A**：
1. 在企业微信后台创建应用
2. 获取 CorpID、Secret、AgentId
3. 填写到 `config_loan_example.yaml`
4. 重命名为 `config.yaml`
5. 运行 `python main.py`

### Q11：如何收集历史聊天记录学习？
**A**：支持三种格式：
1. **JSON格式**：`[{"question":"问题","answer":"回答"}]`
2. **CSV格式**：question,answer 两列
3. **文本格式**：Q: 问题 A: 回答

放入 `data/history/` 目录即可。

### Q12：如何监控效果？
**A**：系统提供：
1. **日志文件**：`logs/loan_auto.log`
2. **咨询统计**：每日咨询量、常见问题排行
3. **意向分析**：高意向客户识别
4. **转化跟踪**：咨询到面谈转化率

## 五、故障排除

### Q13：关屏后仍无响应？
**A**：排查步骤：
1. 检查命令是否执行成功：`pmset -g | grep sleep`
2. 检查网络连接：`ping 8.8.8.8`
3. 检查OpenClaw状态：`openclaw gateway status`
4. 查看日志：`tail -f ~/.openclaw/logs/gateway.log`

### Q14：回复内容不满意？
**A**：
1. 修改 `config_loan_example.yaml` 中的回复内容
2. 或提供更多历史对话数据供系统学习
3. 或联系我调整关键词匹配规则

### Q15：想暂停或卸载？
**A**：
1. **恢复睡眠设置**：
   ```bash
   sudo pmset -a sleep 1 displaysleep 10
   ```
2. **停止OpenClaw**：
   ```bash
   openclaw gateway stop
   ```
3. **删除项目**：删除 `wechat-auto` 文件夹即可
