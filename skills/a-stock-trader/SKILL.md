# A股操盘助手

为爸爸（李志强）定制的A股实盘操盘工具，管理10000元本金。

## 核心功能

### 1. 实时行情监控
- 大盘指数（上证、深证、创业板）
- 个股实时价格、涨跌幅
- 资金流向（主力、北向）

### 2. 技术分析
- K线形态识别
- 均线多头排列/空头排列
- 成交量分析
- 支撑压力位计算

### 3. 自动截图记录
- 使用PyAutoGUI自动截图
- 记录买入/卖出时点
- 保存K线形态

### 4. 语音播报
- 买卖点语音提醒
- 盘中重要信息播报
- 收盘复盘语音总结

### 5. 仓位管理
- 10000元本金分配
- 单票仓位控制（≤50%）
- 止损止盈提醒

## 操盘规则

### 交易时间
- 早盘：9:30-11:30
- 午盘：13:00-15:00
- 集合竞价：9:15-9:25，14:57-15:00

### 风险控制
- 单笔最大亏损：≤300元（3%）
- 单日最大亏损：≤500元（5%）
- 总仓位控制：≤70%
- 单票仓位：≤5000元（50%）

### 买卖点策略
**买入信号：**
- 大盘企稳，个股基本面良好
- 技术指标支持（均线多头排列）
- 成交量放大
- 有明确止损位（-5%或-8%）

**卖出信号：**
- 达到止盈目标（+10%或+15%）
- 跌破止损位
- 基本面恶化
- 大盘系统性风险

## 每日工作流程

### 开盘前（9:00-9:25）
1. 查看隔夜美股、消息面
2. 分析大盘集合竞价
3. 确定当日交易计划
4. 语音播报今日策略

### 盘中（9:30-15:00）
1. 监控持仓股票
2. 寻找买卖点机会
3. 执行交易计划
4. 截图记录关键时点

### 收盘后（15:00-）
1. 复盘当日操作
2. 更新持仓记录
3. 分析盈亏原因
4. 准备次日计划

## 脚本使用

### 获取实时行情
```bash
python scripts/get_realtime_quote.py <股票代码>
```

### 技术分析
```bash
python scripts/technical_analysis.py <股票代码>
```

### 自动截图
```bash
python scripts/capture_screen.py <保存路径>
```

### 语音播报
```bash
python scripts/voice_alert.py "播报内容"
```
