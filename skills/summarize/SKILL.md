# SKILL.md - Summarize

## Description
自动读、自动总结、自动存记忆。3秒内完成网页、PDF或长文档的核心要点提取与总结。

## When to Use
- 阅读长文章需要快速了解要点
- 处理PDF文档
- 总结视频字幕
- 提取关键信息存入记忆

## Usage

### 总结文本
直接提供文本，我会自动总结核心要点。

### 总结网页
提供URL，我会抓取并总结。

### 总结PDF
提供PDF文件路径，我会提取并总结。

## Example

总结新闻：
```
请总结这篇新闻的要点：[URL]
```

总结PDF：
```
请总结这份PDF的核心内容：[文件路径]
```

## Note
使用系统内置的LLM能力实现，无需额外依赖。
