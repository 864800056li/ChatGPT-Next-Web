# TestFlight + iOS App 项目计划

## 目标
创建一个可以通过 TestFlight 分发的 iOS 应用

## 奖励
**+5,000 EXP** (升级大大怪将军!)

## 技术方案

### 方案1: 原生 iOS App (Swift)
- 使用 Swift + SwiftUI
- 需要 Mac + Xcode
- 需要 Apple Developer 账号 ($99/年)
- TestFlight 分发

### 方案2: 跨平台方案 (推荐)
- 使用 React Native / Flutter
- 一套代码同时支持 iOS/Android
- 仍然需要 Apple Developer 账号
- TestFlight 分发

### 方案3: PWA + TestFlight Web App
- 使用 PWA (渐进式Web应用)
- 包装成 iOS App
- 成本最低
- 功能可能受限
