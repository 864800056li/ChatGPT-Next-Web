# A股交易规则 - 大大怪将军学习笔记

## 📅 交易时间
- **早盘**: 9:30 - 11:30
- **午盘**: 13:00 - 15:00
- **集合竞价**: 9:15-9:25（开盘）、14:57-15:00（收盘）
- **周末及节假日**: 休市

## 💰 交易单位
- **最小单位**: 100股 = 1手
- **买入**: 必须是100股的整数倍
- **卖出**: 可以是零股（不足100股）

## 📊 涨跌幅限制
- **普通股票**: ±10%
- **ST股票**: ±5%
- **科创板/创业板**: ±20%
- **新股上市首日**: 无涨跌幅限制（有临时停牌机制）

## 💸 交易费用
| 费用 | 费率 | 说明 |
|------|------|------|
| 佣金 | 0.025%-0.03% | 最低5元，双向收取 |
| 印花税 | 0.05% | 卖出时收取 |
| 过户费 | 0.001% | 双向收取 |

**举例**: 买入10000元股票，费用约 5元佣金 + 0.1元过户费 = 5.1元

## 🎯 T+1交易制度
- **当天买入**: 当天不能卖出（T+1）
- **当天卖出**: 资金可用（买其他股票），但不能提现（T+1到账）

## ⚠️ 风险提示
1. **不要满仓**: 建议保留20-30%现金
2. **止损纪律**: 设置-5%或-8%止损线
3. **分散投资**: 不要把所有钱买一只股票
4. **勿追涨杀跌**: 理性投资，不要情绪化操作

## 📈 大大怪将军操盘原则

### 买入信号
- 大盘企稳，个股基本面良好
- 技术指标支持（均线多头排列）
- 有明确止损位

### 卖出信号
- 达到止盈目标
- 跌破止损位
- 基本面恶化
- 大盘系统性风险

### 每日任务
- [ ] 开盘前：查看隔夜美股、消息面
- [ ] 9:25：观察集合竞价
- [ ] 盘中：监控持仓，执行交易计划
- [ ] 收盘：复盘当日操作，更新持仓记录

## 💼 10000元本金操作方案

### 建议配置
- **单只股票**: 最多3000-5000元（30-50%仓位）
- **留出现金**: 至少3000元备用
- **最多持股**: 2-3只

### 风险控制
- **单笔最大亏损**: 不超过本金的3%（300元）
- **单日最大亏损**: 不超过本金的5%（500元）
- **总仓位控制**: 不超过70%
