# 大大怪下士技能升级记录

## 升级时间
2026-03-14

## 新增能力

### 1. 交易信号分析 (trading_analyzer.py)
- **技术指标计算**：MA、MACD、RSI、KDJ、布林带、ATR
- **信号生成**：基于多指标综合判断买入/卖出/持有信号
- **仓位管理**：自动计算止损位、止盈位
- **置信度评分**：0-100分，量化信号强度

### 2. 模拟盘管理 (paper_trading.py)
- **虚拟交易**：模拟买入/卖出，跟踪成本
- **持仓管理**：实时计算盈亏、收益率
- **交易记录**：完整记录每笔交易及理由
- **组合报告**：生成模拟盘收益报告

### 3. 价格预警系统 (alert_manager.py)
- **多类型预警**：突破、跌破、涨幅、跌幅、放量
- **实时监控**：分析时自动检查是否触发预警
- **预警管理**：添加、删除、列出所有预警

### 4. 统一交易CLI (trading_cli.py)
- 命令行入口整合所有交易功能
- 支持：分析、模拟交易、预警管理

## 使用方式

### 交易信号分析
```
分析茅台的交易信号
贵州茅台技术指标
600519买卖点
```

### 模拟盘操作
```
模拟买入茅台100股@1500
模拟卖出600519
查看模拟盘
```

### 价格预警
```
茅台突破1600提醒我
监控600519跌破1400
列出所有预警
```

## 数据存储
- 模拟盘数据：`~/.openclaw/workspace/paper_trading/`
- 预警数据：`~/.openclaw/workspace/stock_alerts/`
